Welcome to **UseOSM**, a comprehensive initiative to showcase and support the use of OpenStreetMap (OSM) across various domains.
This project was honored as a recipient of the [**2025 OSMF EWG Microgrant**](https://blog.openstreetmap.org/2025/12/14/announcing-the-2025-microgrant-program-grantees/), recognizing its contribution to the OSM community. This repository contains resources, tools, and documentation to help organizations and developers leverage OSM effectively.

---

## Project Overview

### What Is OpenStreetMap?
[OpenStreetMap](https://openstreetmap.com) (OSM) is a global, community-driven geographic database maintained by millions of contributors. It powers applications in:
- Navigation and routing
- Emergency and humanitarian response
- Urban planning and smart cities
- Climate research and environmental protection
- Logistics, mobility, and location intelligence

### What Is UseOSM?
UseOSM is about visibility and appreciation. It highlights organizations, platforms, and tools that use, support, and contribute to OSM.

---

## Repository Structure

This repository is organized as follows:

- **awesome-useosm**: A curated list of companies, organizations, and platforms that use OSM. Includes detailed documentation and contribution guidelines.
- **data**: Contains JSON datasets summarizing OSM changesets and yearly statistics.
- **docs**: Documentation for contributing, proposals, and meeting notes.
- **frontend**: The source code for the UseOSM website, built with Astro. Includes components, layouts, and utilities.
- **scripts**: Python scripts for generating and analyzing OSM statistics.
